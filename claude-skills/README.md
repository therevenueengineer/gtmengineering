# Claude Skills

A collection of custom Claude Code skills for GTM Engineering workflows.

## Skills

| Skill | Description |
|-------|-------------|
| [multi-agent](./multi-agent/) | Orchestrate parallel sub-agent work with automatic context survival and monitoring |

## Usage

Copy the desired skill folder into your project's `.claude/skills/` directory, or reference it directly in your Claude Code configuration.
